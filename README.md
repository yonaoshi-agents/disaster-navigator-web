# NowWay - インバウンド災害避難支援アプリ

## プロジェクト概要

**NowWay** は、訪日外国人旅行者のための災害避難支援Webアプリケーションです。地震などの災害発生時に、言語の壁や防災知識の欠如により適切な行動が取れない外国人旅行者を支援します。

### 解決する課題

- **インバウンド × 災害**という社会課題
  - 日本は地震・災害大国である
  - 訪日外国人は日本語が読めず、防災知識・文化的前提がない
  - 行政の情報にアクセスしづらい
- 災害時に「正しい行動が取れない」ことが
  - 命の危険
  - パニック・混乱
  - 二次被害に直結

### ソリューション

AI/生成AIを活用した多言語対応の災害避難支援システム
- リアルタイムの災害情報提供
- 個別状況に応じた行動指針の提示
- 直感的なUI/UXによる迅速な情報理解

## UI設計コンセプト

### v2: スワイプベースのシンプルUI（最新版）

#### デザイン哲学
- **1画面1アクション** - 緊急時の認知負荷を最小化
- **画面2分割** - 情報と行動を明確に分離
- **スワイプ操作** - 直感的で迷わないインタラクション

#### 画面構成

**上半分: 重要情報エリア**
- 震度、マグニチュード、震源地などの重要情報を大きく表示
- 発生時刻と地域名を明示
- 赤×白のハイコントラスト配色で緊急性を伝達

**下半分: アクションカードエリア**
- ネクストアクションを1つだけ大きく表示
- カード操作:
  - **右スワイプ**: 行動完了 → 次のアクションへ
  - **左スワイプ**: 行動不可能 → 次のアクションへ
- 進捗バーで全体の完了状況を可視化

#### 主要機能

1. **災害情報表示**
   - 震度、マグニチュード、震源地
   - 発生時刻、余震警告
   - 津波警報の有無

2. **ステップバイステップのアクション提示**
   - 頭を保護する
   - テーブルの下に隠れる
   - 安全な場所に移動する
   - 最寄りの避難所を確認する
   - 緊急連絡先に連絡する

3. **多言語対応**
   - English / 日本語の切り替え
   - 右上の言語切り替えボタンで即座に変更可能

4. **進捗管理**
   - 全アクションの進行状況をビジュアル表示
   - 完了時の成功画面

## 認証・課金システム

### サインアップ画面 (`/signup`)

**入力項目:**
- メールアドレス
- パスワード
- 使用言語（English / 日本語 / 中文 / 한국어）

**デザイン特徴:**
- シンプルなフォームレイアウト
- 視認性の高い入力フィールド
- サインインページへの導線

### サインイン画面 (`/signin`)

**入力項目:**
- メールアドレス
- パスワード

**デザイン特徴:**
- ミニマルなフォーム
- パスワード忘れ機能
- サインアップページへの導線

### 課金画面 (`/subscribe`)

**コンセプト:**
旅行前に「災害保険」として加入し、災害発生時にサービスを利用可能にする

**入力項目:**
- 旅行日数（1〜365日）
- プラン選択

**提供プラン:**

#### Basic Plan
- 価格: $9.99/旅行
- 機能:
  - リアルタイム災害アラート
  - 多言語対応（4言語）
  - 基本的な避難ガイド
  - 緊急連絡先リスト

#### Premium Plan（推奨）
- 価格: $19.99/旅行
- 機能:
  - Basic機能すべて
  - AIチャットサポート（24時間対応）
  - パーソナライズされた避難ルート
  - 家族共有機能（最大4名）
  - SMS緊急通知

**デザイン特徴:**
- 2カラムのプラン比較レイアウト
- Premiumプランに「Most Popular」バッジ
- 明確なCTAボタン
- 信頼感のある配色

## 使い方

### 開発環境

1. **プロジェクトのセットアップ**
\`\`\`bash
npm install
npm run dev
\`\`\`

2. **アクセス**
\`\`\`
http://localhost:3000
\`\`\`

### ユーザーフロー

1. **事前登録（旅行前）**
   - `/signup` でアカウント作成
   - `/subscribe` で保険プランに加入
   - 使用言語を設定

2. **災害発生時**
   - アプリを開く
   - 上半分で災害情報を確認
   - 下半分のアクションカードをスワイプしながら行動
   - すべてのアクション完了まで誘導

3. **言語切り替え**
   - 右上の言語ボタンでいつでも切り替え可能
